Metasploit Cheat Sheet

Sunday, Feb 21, 2021

Metasploit Cheat Sheet The Metasploit Project is a computer security project that provides information on vulnerabilities, helping in the development of penetration tests and IDS signatures. Metasploit is a popular tool used by pentest experts. I have prepared a document for you to learn. Metasploit : Search for module: msf > search [regex] Specify and exploit to use: msf > use exploit/[ExploitPath] Specify a Payload to use: msf > set PAYLOAD [PayloadPath] Show options for the current modules: msf > show options Set options: msf > set [Option] [Value] Start exploit: msf > exploit Useful Auxiliary Modules Port Scanner: msf > use auxiliary/scanner/portscan/tcp msf > set RHOSTS 10.
