  • we will be using hashcat to crack the password! first let’s find the mode. we’re looking for SHA-512 We can see that the code for SHA-512 is 1700 which we can use to crack the hash but apparently this does not work. Now, if we recall then in the SSH backdoor code, there was a hardcoded salt involved. We need to add this salt value to the hash as:

    <hash_value>:<salt>
